This is a plugin to insert the VAT ID in the checkout field of WooCommerce and in the view order administration area.

== Description ==

This is a plugin to insert the VAT ID in the checkout field of [WooCommerce](http://www.woothemes.com/woocommerce/) and in the view order administration area.

== Installation ==

1. Upload the entire 'woocommerce-VAT' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
